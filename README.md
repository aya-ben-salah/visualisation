Global YouTube Statistics — Visualisation Project

Ce projet analyse les statistiques des plus grandes chaînes YouTube dans le monde. Il repose sur un dataset contenant plus de 28 variables et couvre plusieurs étapes : nettoyage des données, analyse exploratoire et visualisations avancées.

Préparation des données

Les données ont été nettoyées avant l’analyse :

* Suppression des doublons sur les noms de chaînes
* Remplacement des valeurs manquantes pour les pays par “Unknown”
* Sélection des variables utiles comme Youtuber, Country, channel_type, subscribers, video views, uploads, lowest_yearly_earnings et highest_yearly_earnings
* Création d’une variable earnings_midpoint représentant la moyenne des revenus minimum et maximum
